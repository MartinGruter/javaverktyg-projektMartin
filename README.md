# Javaverktyg och byggmiljöer

Vår applikation är en webshop byggd med Spring Boot.
Applikationen innehåller 4 kategorier:

- Kund
- Order
- Produkt
- Recension

Användare kan skapa, läsa, uppdatera och radera information inom alla kategorier.

## Projektet är byggt med:

- Spring Boot
- SpringData JPA
- Thymeleaf
- Maven
- Docker
- PostgreSQL(Neon)

## Projektet startas genom att

1. klona projektet
   git clone https://github.com/MartinGruter/javaverktyg-projekt

2. Kör huvudklassen JavaverktygProjektApplication

3. Öppna applikationen i webbläsaren http://localhost:8080

## Miljövariabler

För att applikationen ska köras i produktion krävs följande miljövariabler:
**DB_URL** - Anslutnings-URL till PostgreSQL-databas
**DB_USERNAME** - Databasens användarnamn
**DB_PASSWORD** - Databasens lösenord
